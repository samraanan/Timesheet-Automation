# Timesheet Automation

מערכת אוטומטית לעיבוד דוחות שעות ויצירת דוחות Excel מפורטים.

## תכונות עיקריות

- ✅ **פיצול אוטומטי לפי חודשים** - כל חודש מקבל קובץ דוח נפרד
- ✅ **חישוב ק"מ אוטומטי** - מחשב מרחקי נסיעה על בסיס מסלול יומי
- ✅ **פורמט שעות עשרוני** - הצגת שעות כמספרים (9.5 במקום 9:30)
- ✅ **דוחות מרובים** - סיכום לפי לקוח, סיכום מפורט, סיכום עובד, סיכום נסיעות
- ✅ **תמיכה בעברית מלאה** - כולל RTL באקסל
- ✅ **קונפיגורציה גמישה** - ניהול פרויקטים פעילים, מיפוי שמות, והגדרות

## דרישות מערכת

- Python 3.7+
- pandas
- xlsxwriter
- openpyxl

## התקנה

```bash
# התקנת תלויות
pip install pandas xlsxwriter openpyxl
```

## שימוש

### הרצה רגילה (עם GUI)
```bash
python process_timesheets.py
```

הסקריפט יבקש ממך לבחור:
1. קבצי שעות (Excel)
2. קובץ Config.xlsx (אם לא נמצא אוטומטית)

### הרצה אוטומטית (CLI)
```bash
python process_timesheets.py "path/to/timesheet.xlsx"
```

## קובץ Config.xlsx

הקובץ מכיל 3 גיליונות:

### 1. Projects
| Project | Map_To | Active |
|---------|--------|--------|
| אור | אור | כן |
| דרור | דרור | כן |

- **Project**: שם הפרויקט כפי שמופיע בקובץ השעות
- **Map_To**: שם המיקום לחישוב ק"מ (אופציונלי)
- **Active**: האם לכלול בדוח (כן/לא)

### 2. Settings
| Setting | Value |
|---------|-------|
| Home_Location | בית (כפר אלדד) |
| Ignore_Keywords | סה"כ,Total |

### 3. Distances
| מיקום א' | מיקום ב' | מרחק (ק"מ) |
|----------|----------|-------------|
| בית (כפר אלדד) | אור | 25 |
| אור | דרור | 15 |

**הערה:** המערכת בודקת אוטומטית את שני הכיוונים, אז אין צורך להגדיר כל זוג פעמיים.

## הדוחות שנוצרים

הסקריפט יוצר קובץ Excel עם 4+ גיליונות:

1. **סיכום לפי לקוח** - פירוט שעות לפי תאריך ופרויקט
2. **סיכום מפורט** - כולל שעות כניסה/יציאה, הפסקות, סה"כ נטו, ק"מ
3. **סיכום עובד מקוצר** - רק שעות עבודה וק"מ
4. **סיכום נסיעות** - מסלול יומי מפורט עם ק"מ
5. **גיליונות לפי פרויקט** - פירוט מלא לכל פרויקט

## תכונות מתקדמות

### תגית "ללא נסיעה"
אם יש עמודת "תגיות" בקובץ השעות, שורות המסומנות "ללא נסיעה" לא ייכללו בחישוב ק"מ.

### אימות אוטומטי
```bash
python verify_changes.py
```
בודק שהסקריפט עובד תקין ומייצר דוחות נכונים.

## פתרון בעיות

### "Missing distance" warnings
אם אתה רואה אזהרות על מרחקים חסרים:
1. פתח את `Missing_Distances.txt` שנוצר
2. הוסף את הזוגות החסרים לגיליון "Distances" ב-Config.xlsx

### "Permission denied" errors
סגור את קובץ הדוח האקסל אם הוא פתוח.

### בעיות קידוד עברית
הסקריפט מטפל אוטומטית בקידוד UTF-8. אם יש בעיות, בדוק שקבצי הקלט שמורים ב-UTF-8.
